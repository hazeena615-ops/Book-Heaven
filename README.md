# 📚 Book Heaven – Online Book Store

## 📌 Project Overview

Book Heaven is a full-stack MERN-based online bookstore web application developed for learning and placement purposes. The platform allows users to explore, read, buy, and sell books through an interactive and modern user interface.

The application includes a secure Firebase Google Authentication system for user login and an Admin Dashboard for managing books and platform content. Admin users can add new books, edit existing books, and manage the bookstore inventory efficiently.

The project is designed with a responsive frontend and scalable backend architecture using React.js, Node.js, Express.js, MongoDB, and Firebase Authentication.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/hazeena615-ops/Book-Heaven.git
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 3️⃣ Start Backend Server

```bash
npm start
```

---

## 4️⃣ Install Frontend Dependencies

Open another terminal:

```bash
cd mern-client
npm install
```

---

## 5️⃣ Run Frontend

```bash
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file inside the backend folder if needed.

Example:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
